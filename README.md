Project Name : DigiTools

Project Description:
DigiTools is a modern digital marketplace where users can explore and purchase premium digital tools such as AI software, templates, and productivity resources. The platform is designed to provide a smooth user experience with an interactive UI, real-time cart system, and responsive design for all devices.

Use Technology:
i.React.js
ii. JavaScript
iii. Tailwind CSS
iv. DaisyUI
v.React icon
vi.React Toastify

Key Features:
1.Dynamic Cart System:
Users can add products to the cart and instantly see updates in the cart count and total price. Duplicate items are prevented, and items can be removed individually or cleared all at once.

2.Product Browsing & Tabs:
Users can switch between different sections like Products and Cart using tabs. Products are dynamically loaded from JSON data and displayed in a clean card layout.

3.Fully Responsive Design:
The entire application is optimized for mobile, tablet, and desktop devices using Tailwind CSS, ensuring a seamless experience across all screen sizes.

4.Toast Notifications:
Users receive instant feedback (success/error) when adding or removing items using toast notifications.
